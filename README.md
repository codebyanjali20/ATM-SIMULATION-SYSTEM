# ATM Simulation System

A console-based ATM Simulation System developed using Java.

## Features

- PIN Verification
- Balance Inquiry
- Deposit Money
- Withdraw Money
- Menu Driven Interface
- Object-Oriented Programming

## Technologies Used

- Java
- OOP
- Scanner Class

## Project Structure

- Main.java
- ATM.java
- Account.java

## Sample Output

- Login using PIN
- Deposit money
- Withdraw money
- Check balance

## Future Improvements

- Account Number Login
